The English version is below.
在手机上下载Termux,然后输入pkg install git -y然后再根据自己的地区来选择链接下载
中国:
git clone https://gitclone.com/github.com/Wang-Anyi/mc-server-tool.git
然后再输入
bash ~/mc-server-tool/mc-server-tool_zh.sh
等待一下就可以使用了,下次启动还是这个代码(bash ~/mc-server-tool/mc-server-tool_zh.sh)

Download Termux on your phone, then enter `pkg install git -y`, and then choose the download link according to your region. 
International:`git clone https://gitclone.com/github.com/Wang-Anyi/mc-server-tool.git`
Then enter:`bash ~/mc-server-tool/mc-server-tool_en.sh`
Wait a moment and you can use it. The next time you start, use the same command (`bash ~/mc-server-tool/mc-server-tool_en.sh`).
